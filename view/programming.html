Tecnologías FrontEnd y BackEnd

Las tecnologías FrontEnd y el BackEnd son las siguientes:

  • HTML (HyperText Markup Language)
    Es el lenguaje de marcado estándar utilizado para crear y diseñar páginas web. HTML permite definir la estructura y el contenido de una página web mediante el uso de etiquetas y atributos [13].
  • CSS (Cascading Style Sheets)
    Es un lenguaje de hojas de estilo utilizado para describir el aspecto y el formato de una página web escrita en HTML. CSS permite separar la presentación visual de la estructura de la página, lo que hace que el mantenimiento y la actualización del sitio sean más sencillos [14].
  • JavaScript
    Es un lenguaje de programación interpretado utilizado para crear interactividad en páginas web. JavaScript permite crear efectos dinámicos, validar formularios, modificar el contenido de una página web en tiempo real, entre otras funcionalidades [15].
  • PHP (Hypertext Preprocessor)
    Es un lenguaje de programación utilizado principalmente para desarrollar aplicaciones web del lado del servidor. PHP permite generar contenido dinámico, interactuar con bases de datos, gestionar cookies y sesiones, entre otras funcionalidades [16].
  • MySQL
    Es un sistema de gestión de bases de datos relacional utilizado para almacenar y gestionar datos de manera eficiente. MySQL es uno de los sistemas de gestión de bases de datos más utilizados en el mundo y es compatible con la mayoría de los lenguajes de programación y sistemas operativos [17].
